# Scanner
Считывает входные данные и разбивает их на строки или слова
```go
input := bufio.NewScanner(inp)
// inp == 
// os.Stdin
// os.File
```
Каждый вызов `input.Scan()` считывает очередную строку и удаляет завершающий символ новой строки.
Результат можно получить путем вызова `input.Text()`.

Функция Scan возвращает значение `true` , если строка считана и доступна, и значение `false`, если входные данные исчерпаны.
