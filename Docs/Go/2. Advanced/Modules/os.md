# Open
```go
f, err := os.Open(arg)
```
Функция `os.Open` возвращает два значения. Первое из них является открытым файлом (`*os.File`). Второй результат `os.Open` — значение встроенного типа `error` . Если `err` равно специальному встроенному значению `nil` , файл открыт успешно. ^37631f
