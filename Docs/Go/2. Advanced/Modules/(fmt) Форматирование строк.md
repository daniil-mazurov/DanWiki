|            |                                                                   |
| ---------- | ----------------------------------------------------------------- |
| %d         | Десятичное целое                                                  |
| %x, %o, %b | Целое в шестнадцатеричном, восьмеричном и двоичном представлениях |
| %f, %g, %e | Числа с плавающей точкой: 3.141593 3.141592653589793 3.141593е+00 |
| %t         | Булево значение: `true` или `false`                               |
| %c         | Символ Unicode                                                    |
| %s         | Строка                                                            |
| %q         | Выводит в кавычках строку типа "abc" или символ типа ' с '        |
| %v         | Любое значение в естественном формате                             |
| %T         | Тип любого значения                                               |
| %%         | Символ процента (не требует операнда)                             |
# Печать в терминал
- `Printf` - печатает отформатированный текст с вербами и аргументами
- `Print` - печатает текст без форматирования
- `Println` - печатает отформатированный текст + добавляет символ перевода строки в конце
# Печать в поток данных
- `Fprintf`
- `Fprint`
- `Fprintln`
# Печать в строку
- `Sprintf`
- `Sprint`
- `Sprintln`
# Примеры использования
```go
name := "Tatyana"
age := 25
fmt.Printf("Hello, %v! You are %v years old\n", name, age)
fmt.Println("Hello,", name, "! You are", age, "years old")
```
