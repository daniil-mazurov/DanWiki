Используется:
- **Указатели** - когда не ссылаются на объект
```go
var p *int = nil
```
- **Интерфейсы** - когда не содержат конкретного значения
```go
var err error = nil
```
- **Срезы** - неинициализированный срез 
```go
var s []int = nil
```
- **Мапы** - неинициализированная мапа
```go
var m map[string]int = nil
```
- **Каналы** - неинициализированный канал
```go
var ch chan int = nil
```
- **Функции** - неинициализированная функция
```go
var f func() = nil
```

# Важные нюансы:

1. В Go `nil` - это не объект, а зарезервированное ключевое слово
2. `nil` в Go строго типизирован - `nil` указателя на `int` не равен `nil` интерфейса
3. В Go нужно явно проверять на `nil` перед использованием
