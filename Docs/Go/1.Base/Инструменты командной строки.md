- **go build** - компилирует проект и создает исполняемый бинарный файл
- **go build -race** - проверка наличия проблем с параллеьным запуском
- **go run** - запуск проекта напрямую, без создания исполнямого файла
- **go mod** - управление зависимостями
- **go mod tidy** - обновление зависимостей
- **go test** - запуск тестов
- **go fmt** - форматирование файлов исходного кода (обычно выполняется автоматически в IDE)
