# Стандартный цикл
```go
for инициализация; условие; последействие { 
	// нуль или несколько инструкций 
}
```

```go
for i := 0; i < 10; i++ {
	fmt.Println(i)
}
```
Любая из перечисленных частей может быть опущена. При отсутствии как инициализации, так и последействия можно опустить точки с запятыми: 
```go
// Традиционный цикл "while" 
for condition { 
	// ... 
	// break
	// continue
}
```
# Цикл с инкрементом внутри тела
```go
i := 0
for i < 10 {
	fmt.Println(i)
	i++
}
```
# Бесконечный цикл c прерыванием (break)
```go
somethingHappened := true
	for {
		if somethingHappened {
			break
		}
	}
```
# Цикл с пропуском части элементов
```go
for i := 0; i < 10; i++ {
	if i%2 == 0 {
		continue
	}
	fmt.Println(i)
}
```
