По умолчанию при передаче данных в метод создается их локальная копия. Если объем данных велик - получает проблемы с производительностью/ресурсами.  

Для решения этой (и других) проблем используются **указатели**.  
С помощью него можно передать в метод сам оригинальный объект (точнее ссылку на него).  
  
Какие символы используются для работы с указателями:  

1. **\*** оператор разыменования / астерикс. Два вида применения: 
	1) Обозначает указатель 
	2) Забирает значение переменной по указателю
2. **&** создает указатель на переменную
# Создание указателя
```go
myVar := 10
var myPtr *int
myPtr = &myVar // 0xc000104020

// Создаем указатель (краткая форма)
myAnotherPtr := &myVar
```
# Получаем значение переменной из указателя
```go
value := *myPtr // 10
```
# Использование указателя в функции
```go
func myInc(x *int) {
	*x++
}

func main() {
	myVar := 10 // 10
	myInc(&myVar) // 11
	myInc(&myVar) // 12
}
```
# Разница между `*` в объявлении типа и в выражении

1. В **объявлении типа** - означает тип указателя:
```go
var p *int  // p - указатель на int
```
2. В **выражении** - означает разыменование указателя:
```go
value := *p  // Получить значение, на которое указывает p
```
# Разница между `var p *int` и `var p int` в Go

Основное отличие заключается в том, что первое объявление создает **указатель на целое число**, а второе - **само целое число**. 
## 1. `var p int`
- Создает переменную типа `int` (целое число)
- Выделяет память для хранения целого числа
- Инициализируется нулевым значением (0)
```go
var p int        // p имеет тип int
p = 42           // присваиваем значение
fmt.Println(p)   // Вывод: 42
```
## 2. `var p *int`
- Создает переменную типа `*int` (указатель на целое число)
- Инициализируется `nil` (не указывает ни на что)
- Требует дополнительного выделения памяти для хранения самого числа
```go
var p *int       // p имеет тип *int (указатель на int)
fmt.Println(p)   // Вывод: <nil> (нулевое значение для указателей)

// Для использования нужно:
// 1. Создать int переменную и взять её адрес
value := 42
p = &value       // p теперь указывает на value

// ИЛИ 2. Выделить память с помощью new()
p = new(int)     // выделяет память для int, инициализирует 0
*p = 42          // разыменовываем указатель и присваиваем значение
```

## Когда что использовать?

### Используйте `var p int` когда:

- Нужно просто целое число
- Работаете с базовыми значениями
- Не нужно передавать значение для модификации в функциях

### Используйте `var p *int` когда:

- Нужно передать значение в функцию для модификации
- Работаете с опциональными значениями (может быть nil)
- Оптимизируете работу с большими структурами (чтобы избежать копирования)
